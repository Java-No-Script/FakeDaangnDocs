# 디자인 토큰

디자인 토큰은 제품 전반의 시각 언어를 코드로 표현한 최소 단위의 값입니다.

## 범주

- 컬러: `--color-bg`, `--color-fg`, `--color-primary` 등
- 타이포: `--font-size-sm/md/lg`, `--line-height-base`
- 스페이싱: `--space-2/4/8/12/16` 등 규칙적 간격 스케일
- 라운드/보더: `--radius-sm/md/lg`, `--border-1`
- 그림자: `--shadow-sm/md/lg`

## 사용 예시

```css
:root {
  --color-primary: #ff6f0f;
  --space-8: 8px;
}
.button-primary {
  background: var(--color-primary);
  padding: var(--space-8) var(--space-8);
  border-radius: 8px;
}
```
