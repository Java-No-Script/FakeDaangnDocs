# Input

<Callout emoji="⌨️">
  문자 입력 필드입니다. `label`, `helpText`, `errorText`로 사용성을 높입니다.
</Callout>

## 빠른 시작

```tsx
import { Input } from "@/components";

export default function Example() {
  return (
    <>
      <Input
        id="name"
        label="이름"
        placeholder="성함을 입력하세요"
        helpText="실명 기준"
      />
      <Input
        id="email"
        label="이메일"
        type="email"
        errorText="형식을 확인해 주세요"
      />
      <Input id="nick" label="닉네임" size="sm" placeholder="2~12자" />
    </>
  );
}
```

## 사용 방법

### 이벤트를 주는 방법(Controlled)

```tsx
import { useState } from "react";
import { Input } from "@/components";

export default function Controlled() {
  const [value, setValue] = useState("");
  return (
    <Input
      id="title"
      label="제목"
      value={value}
      onChange={(e) => setValue(e.target.value)}
    />
  );
}
```

### 유효성 검사(즉시/블러/서버)

```tsx
// 즉시 검증 (길이)
const [v, setV] = useState('')
const error = v.length < 2 ? '2자 이상 입력하세요' : undefined
<Input id="nick" label="닉네임" value={v} onChange={(e)=>setV(e.target.value)} errorText={error} />
```

```tsx
// 블러 시 검증
const [error, setError] = useState<string | undefined>()
<Input id="email" label="이메일" onBlur={(e)=> setError(/@/.test(e.target.value) ? undefined : '형식을 확인해 주세요')} errorText={error} />
```

```tsx
// 서버 검증 (중복 확인 예시)
const [error, setError] = useState<string | undefined>();
const onBlur = async (e: React.FocusEvent<HTMLInputElement>) => {
  const ok = await fetch(
    `/api/check?nick=${encodeURIComponent(e.target.value)}`
  ).then((r) => r.ok);
  setError(ok ? undefined : "이미 사용 중입니다");
};
<Input id="nick" label="닉네임" onBlur={onBlur} errorText={error} />;
```

### 테마/상태 색 변경

- 전역 CSS 변수로 상태 테마를 조정할 수 있습니다(예: 에러 테두리 색).

```css
:root {
  --color-primary: #ff6f0f;
}
/* 영역 한정 테마 */
[data-theme="brandB"] {
  --color-primary: #0a84ff;
}
```

```tsx
<div data-theme="brandB">
  <Input id="name" label="이름" />
</div>
```
