# Button

<Callout emoji="🔘">
  행동(액션)을 트리거하는 기본 컴포넌트입니다. `variant`와 `size`로 일관된
  스타일을 제공합니다.
</Callout>

## 빠른 시작

```tsx
import { Button } from "@/components";

export default function Example() {
  return (
    <>
      <Button variant="primary">저장</Button>
      <Button variant="secondary" size="sm">
        취소
      </Button>
      <Button variant="text" size="lg" disabled>
        불가
      </Button>
    </>
  );
}
```

## 사용 방법

### 이벤트를 주는 방법

```tsx
import { Button } from "@/components";

function SaveButton() {
  const handleSave = async () => {
    // 비동기 처리 예시
    await submitForm();
  };
  return <Button onClick={handleSave}>저장</Button>;
}
```

- 비동기 작업 동안 중복 클릭을 막고 싶다면 상태로 `disabled`를 제어하세요.

```tsx
const [loading, setLoading] = useState(false);
const onClick = async () => {
  setLoading(true);
  try {
    await run();
  } finally {
    setLoading(false);
  }
};
<Button onClick={onClick} disabled={loading}>
  {loading ? "처리 중…" : "저장"}
</Button>;
```

### 사이즈를 변경하는 방법

```tsx
<Button size="sm">작게</Button>
<Button size="md">보통</Button>
<Button size="lg">크게</Button>
```

- `sm`는 보조 UI, `md`는 기본, `lg`는 모바일 주 액션/히어로 등에 권장.

### 테마를 변경하는 방법

- 전역 CSS 변수로 브랜드 색을 바꿀 수 있습니다. 이 컴포넌트는 `--color-primary`를 사용합니다.

```css
:root {
  --color-primary: #ff6f0f;
} /* 기본값 */
/* 다크/브랜드 테마 예시 */
[data-theme="brandB"] {
  --color-primary: #0a84ff;
}
```

```tsx
// 특정 영역만 바꾸고 싶다면 래퍼에 data-theme를 부여
<div data-theme="brandB">
  <Button>브랜드 B 색상</Button>
</div>
```

- 개별 버튼만 미세 조정이 필요하면 `style`로 보더/배경을 덮어쓸 수 있습니다.

```tsx
<Button style={{ background: "#222", color: "#fff", borderColor: "#222" }}>
  커스텀
</Button>
```
