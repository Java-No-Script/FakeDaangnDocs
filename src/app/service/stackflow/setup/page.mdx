# 세팅

<Callout emoji="⚙️">몇 단계만으로 기본 흐름을 구성할 수 있습니다.</Callout>

## 설치

```bash
npm i @stackflow/core @stackflow/react
```

## 초기화

```tsx
import { createStackflow } from "@stackflow/react";

const { Stack, useNavigator } = createStackflow({
  screens: {
    Home: () => <div>Home</div>,
    Detail: () => <div>Detail</div>,
  },
});
```

## 사용(푸시/팝)

```tsx
function HomeScreen() {
  const nav = useNavigator();
  return <button onClick={() => nav.push("Detail", { id: 1 })}>상세</button>;
}
```
