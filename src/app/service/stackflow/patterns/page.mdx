# 패턴

## 자주 묻는 시나리오

<Tabs items={["중첩 스택", "모달", "가드/백버튼"]}>
  <Tabs.Tab>
    <Callout emoji="🧭">상세 → 리뷰 작성처럼, 상세 내부에서 추가 흐름이 필요한 경우.</Callout>

    ```tsx
    const { Stack } = createStackflow({ screens: { Detail, ReviewForm } })
    // Detail 내부에서
    const nav = useNavigator()
    nav.push('ReviewForm')
    ```

  </Tabs.Tab>
  <Tabs.Tab>
    <Callout emoji="🪟">모달은 같은 스택에 올리되, UI만 겹치게 표현합니다.</Callout>

    ```tsx
    nav.push('PaymentModal', { amount: 10000 })
    // 완료 후
    nav.pop()
    ```

  </Tabs.Tab>
  <Tabs.Tab>
    <Callout emoji="🛡️">폼 미저장 시 이탈 방지, 백버튼 가드 예시.</Callout>

    ```tsx
    useBeforePop(() => {
      if (dirty) return confirm('변경 사항이 저장되지 않았습니다. 떠나시겠어요?')
      return true
    })
    ```

  </Tabs.Tab>
</Tabs>
