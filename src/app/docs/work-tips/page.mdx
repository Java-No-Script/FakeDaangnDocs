# 업무 팁 모음

## 캘린더/시간 관리

- 포커스 타임을 주 2–3회 2시간 블록으로 예약하고, 알림을 일괄 차단합니다.
- 리뷰/1:1/정기회의는 고정 슬롯으로 운영해 컨텍스트 스위칭을 최소화합니다.

## 문서화/기록

- 결정은 문서로 남기고, 링크를 채널 쓰레드 첫 댓글에 고정합니다.
- 요약 → 근거 → 옵션 → 결정 구조로 작성하여 추적 가능성을 높입니다.

## PR/리뷰

- 작은 단위로 자주 올리고, PR 본문에 배경/변경/리스크/테스트 항목을 명시합니다.
- 리뷰어는 승인/수정요청/질문을 구분하여 남기고, SLA(24h 내 1차 피드백)를 지킵니다.

## 협업

- 의견 충돌은 문제 정의와 성공 기준부터 정렬합니다.
- 회의는 의제/목표/결정 예정자 명시, 회의록은 24h 내 공유합니다.
