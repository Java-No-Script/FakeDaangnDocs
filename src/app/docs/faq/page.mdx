# 자주 묻는 질문

## 첫날/첫 주

- 출근 시간은? — 코어타임 없음. 첫날 10:00까지 도착 권장.
- 오피스 출입 — 모바일 사원증은 IT에서 발급. 임시 카드 수령 가능.
- 장비 배포 — 맥북 + 보조 모니터, 필요 시 아이패드/테스트폰 신청 가능.

## 장비/IT

- 추가 소프트웨어 설치 — 셀프서비스 포털 우선, 미등록 소프트웨어는 `#IT운영지원팀` 문의.
- 저장소 접근 권한 — GitHub/데이터 웨어하우스 권한은 매니저 승인 후 요청.
- 계정 잠김/2FA — Okta/SSO 기준, 분실 시 즉시 `#보안-문의`.

## 재무/정산

- 법인카드 발급 — 팀 리더 추천 후 재무 승인. 개인 선결제는 한도 내 월 1회 일괄 정산.
- 영수증 분실 — 카드사 사용내역 + 거래명세서로 대체 가능(사유서 포함).

## 휴가/경조사

- 반반차 사용 — 2시간 단위, 동일 반일과 중복 사용 불가.
- 경조사 증빙 — 청첩장/부고장 스캔본 제출. 내부 양식 제공.

## 보안

- 외부 반출 데이터 — 민감정보는 반출 금지. 불가피 시 CISO 승인 및 암호화 필요.
- 기기 분실 — 즉시 IT/보안에 신고, 원격 잠금·와이프 진행.
