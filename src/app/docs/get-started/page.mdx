# Get Started: 첫 주 체크리스트

- [서비스](/service) — 기능군/용어/플로우 큰 그림
- [팀문화](/culture) — 협업 방식·의사결정 원칙
- [온보딩](/docs/onboarding) — 전체 흐름 요약

## 2. 제품 투어 (60–90분)

<Steps>

### 중고거래 흐름 파악

구매/판매 전 과정을 앱 기준으로 따라가며 용어를 기록합니다.
게시/발행/쿠폰/후기 등 핵심 이벤트를 지도처럼 정리합니다.

### 동네생활/동네가게 이해

비즈/브랜드 프로필 생성 흐름과 지역 타게팅 개념을 익힙니다.

### 비즈니스/광고 살펴보기

비즈/브랜드 프로필 생성 흐름과 지역 타게팅 개념을 익힙니다.

</Steps>

## 3. 신뢰/안전 원칙 (30분)

- 동네 인증, 매너온도, ML 분석의 역할 메모
- 허위/불법 매물 감시 사례 찾아보기

## 4. 협업 준비 (30–60분)

- 팀 채널·회의 캘린더 합류 및 일정 블로킹
- PR/리뷰·문서화·이슈 트래킹 규칙 확인
- 사내 용어집(있다면) 즐겨찾기 추가
