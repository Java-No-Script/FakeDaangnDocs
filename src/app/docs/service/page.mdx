# 서비스 개요

## 디자인 시스템(Seed Design)

<Callout emoji="🌱">
  Seed Design은 당근의 디자인 시스템으로, 일관된 UX를 위한
  토큰·컴포넌트·가이드를 제공합니다.
</Callout>

## 스택 플로우(예시)

- 디자인 → 토큰 정의 → 컴포넌트 구현 → 문서/데모 → 배포 → 소비(제품)
- 변경은 PR로 제안하고, 영향 범위/마이그레이션 가이드를 함께 제공합니다.

<Callout type="info" emoji="🧭">
  자세한 내용은 좌측 바의 디자인 시스템 문서에서 원칙·토큰·컴포넌트를 순서대로
  확인하세요.
</Callout>
